Claude Conversation Exporter Bookmarklet

Export your Claude AI conversations with a single click! This bookmarklet allows you to download your conversations in both Markdown and JSON formats without needing to open the browser console.

Drag this button to your bookmarks bar:

Export Claude Conversation

Can't drag? Right-click and add to bookmarks manually.

How to Install

  1. Make sure your bookmarks bar is visible:
    • Chrome/Edge: Press Ctrl+Shift+B (Windows/Linux) or Cmd+Shift+B (Mac)
    • Firefox: Press Ctrl+B (Windows/Linux) or Cmd+B (Mac)
    • Safari: View ‚Üí Show Bookmarks Bar
  2. Drag the blue "Export Claude Conversation" button above to your bookmarks bar
  3. If dragging doesn't work, right-click the button and select "Bookmark This Link" or "Add to Bookmarks"

How to Use

  1. Open any Claude conversation in your browser
  2. Click the "Export Claude Conversation" bookmarklet in your bookmarks bar
  3. The script will automatically download two files:
    • [conversation-name].md - Formatted Markdown version
    • [conversation-name].json - Raw JSON data
  4. Check your browser's download folder for the exported files

Features

  • Dynamic file naming based on conversation title
  • Exports both Markdown and JSON formats
  • Includes all message content, thinking sections, and tool usage
  • Works entirely in your browser - no data sent to external servers
  • Automatic organization ID detection
  • One-click export without opening developer console
  • Visual notifications showing export progress and status
  • Detailed console logs with organization ID, conversation ID, and metadata

Troubleshooting

  • Bookmarklet doesn't work: Make sure you're on a Claude conversation page (URL should contain /chat/)
  • No downloads: Check your browser's download settings and popup blocker
  • Organization ID prompt: If prompted, you can find your organization ID in the browser's developer console or contact your Claude administrator
